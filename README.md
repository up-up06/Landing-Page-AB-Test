# Landing-Page-AB-Test
Data source: https://mavenanalytics.io/data-playground/toy-store-e-commerce-database

**Описание данных**  
E-Commerce база данных для онлайн-магазина Maven Fuzzy Factory, продающего плюшевых мишек. Она включает подробные маркетинговые данные о сессиях на сайте и просмотрах страниц пользователями, а также об их заказах и возвратах. Идеально подходит для анализа и оптимизации маркетинговых каналов, измерения и тестирования эффективности конверсии сайта, а также для понимания влияния запуска новых продуктов.

**Задание**  
Вы работаете маркетинговым аналитиком в интернет-магазине, продающим детские игрушки. Менеджер сайта заметил необычно высокий показатель bounce rate на главной странице (homepage) - 60%, и принял решение протестировать новую лендинговую страницу для определённого сегмента трафика.  
Ваша задача проанализировать результаты A/B-теста, чтобы определить, привела ли новая целевая страница к статистически значимому улучшению показателю bounce rate.

**Цели**  
1. Отфильтровать трафик веб-сайта
2. Рассчитать bounce rate
3. Проверить значимость

**Рекомендуемый анализ**  
- Какой тренд посещений сайта и количества заказов?
- Каков коэффициент конверсии session-to-order? Какой у него тренд?
- Какие маркетинговые каналы оказались наиболее успешными?
- Как изменился коэфициент выручки с каждого заказа (RPO)? Коэфициент выручки за сессию?
